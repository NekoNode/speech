# Neko Speech

使用Neko Speech为您的网站添加朗读内容功能：

```html
<div id="nekospeech"></div>
<script src="https://speech.cdn.nekonode.com/nekospeech.js"></script>
```

项目面向个人博客等用途免费，发送邮件至speech@nekonode.com以开通服务。
